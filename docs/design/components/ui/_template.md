## 5. 担当しない責務

以下は `Xxx` の責務に含めない。

- 業務ロジックの管理
- API通信
- 画面固有の状態管理
- 画面遷移処理

これらは画面側、上位コンポーネント、または専用コンポーネントで対応する。
