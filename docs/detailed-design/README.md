# 詳細設計

本ディレクトリでは、Taskuest v1における各画面の詳細設計を管理する。

## 対象範囲

- タスク管理
- プロジェクト管理
- メンバー管理
- ユーザー設定

※ v1スコープに基づく

## 画面一覧

- タスク一覧画面: task-list.md
- プロジェクト一覧画面: project-list.md
- プロジェクト詳細画面: project-detail.md
- メンバー管理画面: member-management.md
- ユーザー設定画面: user-settings.md

## 運用ルール

- 各画面ごとに1ファイルで管理する
- 詳細設計は対応する feature ブランチで作成する
- 設計と実装は同一ブランチで進行する
